# Large Project

FitTrack is a full-stack fitness tracking web application that allows users to browse exercises, plan and log workout sessions, and visualize their progress over time through interactive charts.

## Features
- Exercise library
- Workout plan builder and active session tracking
- History & charts
- Auth: Login & register with email verification and password reset option
